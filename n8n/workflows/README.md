# n8n Workflows

Export your n8n workflows as JSON files and commit them here for version control.

## Importing

1. Open n8n editor (http://localhost:5678 by default).
2. Go to Workflows → Import from File.
3. Select the desired `.json` file from this directory.

## Exporting

1. Open the workflow in n8n editor.
2. Go to Workflows → Export → Export to File.
3. Save the JSON file in this directory with a descriptive name.

## Workflow Naming Convention

Use the same names as the automation jobs:

- `post-free-this-week.json`
- `post-crazy-discounts.json`
- `post-top-this-week.json`
- `post-major-updates.json`
- `post-coming-soon.json`
- `calendar-sync.json`
- `calendar-notify.json`
